Rodando o projeto na máquina local

**Execução do projeto:**

**Docker - Na pasta do projeto execute:**
```
docker-compose up
```
**Maven - Na pasta do projeto execute:**

```
1 - mvn clean package
2 - mvn install
3 - mvn spring-boot:run -Dspring-boot.run.profiles=dev
```

**Acesse:**

```
http://localhost:8080
```
